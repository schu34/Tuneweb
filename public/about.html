What is This?

Tuneweb is an app to help you discover new music. When you type the name of a band or musician into the search bar, it will show you a map of related artists. If a line connects two artists, it means they are directly related

How does it work?

When you click the "explore" button, the name of the band you typed in gets sent to a server. The server queries last.fm for the top 5 artists related to what you typed in, then for each of those 5 artists, does the same thing. Now we have a fetched a total of 30 artists, but it's pretty likely that we have some duplicates. The server goes through and removes duplicates from the list, but preserves the relationships between bands. once duplicates have been removed, the server sends the data back to your browser where d3.js is used to create the beautiful visualization. If you're interested in all the nitty gritty details you can check out the source on github
